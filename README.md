# React Boilerplate

Starting code for a react app. Contains:

- Redux Store
- Redux Persist
- React Templates
- Webpack / Babel / Eslint configs
- CSS Bootstrap integration
- Bin folder for scripts
- Basic on-page SEO (OpenGraph and Twitter)
- Favicon

## Start

```bash
npm install
npm run watch
```

## Build

```bash
npm install
npm run build
```
